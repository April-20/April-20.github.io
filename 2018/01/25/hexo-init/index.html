Hexo 从零开始搭建个人博客

发表于 2018-01-25

因为本人平时喜欢做笔记，看到什么学到什么只要有所收获有所感悟就会记下来，到处写到处放，零零碎碎的不易寻找翻看，所以想搭建一个自己的博客网站。接下来，本文将使用hexo从零开始搭建个人博客。

安装 node.js

Hexo是基于nodeJS环境的静态博客，所以第一步需安装nodJS。本文以windows环境安装nodeJS为例，首先登录node.js官网，选择适合自己的版本进行下载，（LTS为长期支持版，Current为当前最新版）。

github账户的注册和配置

第一步：Github注册

打开github官网，在注册表单里分别输入自己的用户名、邮箱、密码后提交，然后前往刚刚填写的邮箱，点开github发送的注册确认邮件，点击确认注册，一定要确认注册！否则无法使用gh-pages！

第二步：创建代码库

登录之后，点击页面右上角的加号，选择New repository新建代码库。进入代码库创建页面，在Repository name 下填写 yourname.github.io( 注意此处的 yourname 要与你的 github用户名 一致，其它名称无效 )，每一个github账户能且只能创建一个这样可以直接使用域名访问的仓库。

比如我的github用户名为：April-20，此处的Repository name就为：April-20.github.io

第三步：添加ssh密钥

点击页面右上角头像 > Settings > SSH and GPG keys > New SSH key

在该页面输入title（自己随便取），key 的内容就是前面使用 git 生成的 id_rsa.pub 文件的内容，直接粘贴复制就行了。
