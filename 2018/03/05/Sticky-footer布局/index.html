Sticky footer布局

发表于 2018-03-05 | 分类于 CSS

概念： 如果页面内容不够长的时候，页脚块粘贴在视窗底部；如果内容足够长时，页脚块会被内容向下推送。

父级要求fixed

方法一：Flex布局

利用flex布局对视窗高度进行分割，footer的flex设为0，这样footer获得其固有的高度，content的flex设为1，这样它会充满除去footer的其他部分。

方法二：在content外面添加一个wrapper

方法三：calc函数

利用calc函数设置main块的最小高度，其中100vh为屏幕的可视高度，需要注意的是，运算符前后都需要保留一个空格。

父级不要求fixed
