call、apply、bind

发表于 2018-02-02 | 分类于 JavaScript

call/apply/bind方法的来源

call、apply、bind这三个方法都是继承自Function.prototype中的，属于实例方法。

普通对象、函数、数组都继承了Function.prototype对象中的这三个方法，所以这三个方法都可以在对象，数组，函数中使用。

call/apply/bind方法的异同

  • 其作用都是手动设置this的指向（即函数执行时所在的作用域），然后在所指定的作用域中，调用该函数。
  • 它们的第一个参数都为this将要指向的对象。（当第一个参数为null或undefined或this时，会自动将函数内部的this对象指向全局环境）。
  • bind和call的传参是以一个一个的形式传递，apply是以数组的形式传递。
  • call和apply会立即执行函数，bind不会立即执行函数，而是返回一个新函数。
